       173 +  ## Water Die Special Handling
       174 +  
       175 +  Water die special handling has been implemented in code. 
           + When a player selects a water die (building die face 
           + "Water"), they must choose a building type (Industrial, 
           + Residential, Commercial, School, Park) for the shape. The 
           + code now supports this with the following changes:
       176 +  
       177 +  1. **DiceManager**: Added `waterDieChosenBuildingType` 
           + field and methods to get/set chosen building type.
       178 +  2. **ShapeManager**: Updated to use 
           + `GetBuildingTypeForShape()` which returns chosen building 
           + type for water die.
       179 +  3. **GameManager**: Updated star awarding to use selected 
           + dice faces (supports water die double rolls).
       180 +  4. **DiceUIManager**: Added UI fields for water die panel 
           + and building type buttons.
       181 +  
       182 +  ### Steps to Set Up Water Die UI in Unity Editor:
       183 +  
       184 +  1. **Create Water Die Panel**:
       185 +     - In the Canvas, create a new Panel child (GameObject ‚Üí 
           + UI ‚Üí Panel). Name it "WaterDiePanel".
       186 +     - Position it appropriately (e.g., centered over the 
           + dice area).
       187 +     - Add a Vertical Layout Group to arrange buttons neatly.
       188 +     - Add a Text header (optional) "Choose Building Type".
       189 +  
       190 +  2. **Create Building Type Buttons**:
       191 +     - Inside WaterDiePanel, create 5 Buttons (GameObject ‚Üí 
           + UI ‚Üí Button).
       192 +     - Name them: "IndustrialButton", "ResidentialButton", 
           + "CommercialButton", "SchoolButton", "ParkButton".
       193 +     - For each button, set the button text to the building 
           + type name (or use icons later).
       194 +     - Style as desired.
       195 +  
       196 +  3. **Assign UI Elements to DiceUIManager**:
       197 +     - Select the GameObject with `DiceUIManager` script.
       198 +     - In the Inspector, find the "Water Die UI" section.
       199 +     - Drag the WaterDiePanel GameObject to the "Water Die 
           + Panel" field.
       200 +     - Drag each button to the corresponding field 
           + (Industrial Button, etc.).
       201 +  
       202 +  4. **Testing**:
       203 +     - Enter Play Mode.
       204 +     - Roll dice until a water die appears (face "Water").
       205 +     - Click the water die (building dice row). The water die
           +  panel should appear.
       206 +     - Click one of the building type buttons. The panel may 
           + stay visible (optional).
       207 +     - Create a shape (click on grid). The shape should have 
           + the chosen building type color.
       208 +     - Place the shape adjacent to a river tile (water die 
           + exception). Validation should pass.
       209 +     - If water die is a double face (appears twice) and 
           + selected, star should be awarded on placement.
       210 +  
       211 +  ### Validation Rules:
       212 +  - Water die selected ‚Üí `waterDieUsedThisTurn` flag set 
           + true.
       213 +  - Shape must be adjacent to river tile (orthogonal 
           + adjacency).
       214 +  - Bypasses first-turn and subsequent-turn adjacency rules.
       215 +  - Player must choose a building type before shape can be 
           + created (otherwise shape creation fails with log message).
       216 +  
       217 +  ### Notes:
       218 +  - The water die panel automatically shows/hides based on 
           + water die selection.
       219 +  - Chosen building type persists while water die remains 
           + selected.
       220 +  - When dice are rolled (new turn), chosen building type is 
           + cleared.
       221 +  - Star awarding now uses selected dice faces (not shape 
           + building type), so water die double rolls correctly award 
           + stars.
